Web Design

KTU 2024 Detailed Syllabus

Module 1

  • Introduction to Internet: Internet protocol - Overview of TCP/IP and its services - URLs - Domain names - Protocols - HTTP, HTTPS, FTP, Telnet, Email, MIME.
  • World Wide Web (WWW): Web browser and Web server - Web publishing - Web hosting - Types of hosting - Static and Dynamic websites - Search Engines - Concepts of Web 2.0.
  • HTML: HTML5 tags and formatting - Lists - Tables - Forms - Input types - Frames - Embedded elements - Audio and Video tags.
  • CSS: Inline, Internal and External CSS - Classes and IDs - Font - Color - Margins - Padding - Background - Border - Box model - Positioning - Floating - Navigation bar - Layout - Grid and Flexbox.

Module 2

  • CSS3: Rounded corners - Shadow - Gradients - Transitions - Animations - Transform - Responsive Web Design - Media Queries.
  • Bootstrap: Introduction - Containers - Grid system - Typography - Colors - Tables - Images - Jumbotron - Alerts - Buttons - Badges - Progress bars - Spinners - Pagination - List group - Cards - Dropdowns - Collapse - Navbar - Forms - Carousel - Modal.
  • Introduction to UI design - Principles of good UI design - Understanding color theory - Typography - Layout and alignment - Accessibility.

Module 3

  • JavaScript: Introduction - Syntax - Variables - Operators - Functions - Events - Conditionals - Loops - Arrays - Objects - Strings - Numbers - Dates - Math - RegExp.
  • DOM manipulation - Changing HTML and CSS - Adding and removing elements - Handling events.
  • Form validation - Built-in validation methods - Creating custom validation.
  • jQuery: Introduction - Syntax - Selectors - Events - Effects - Callback functions - DOM manipulation.

Module 4

  • Web Hosting: Domain registration - Web hosting services - DNS - Hosting a website using cPanel - FTP and FileZilla.
  • SEO: Introduction - Importance - On-page and Off-page SEO techniques.
  • Web Accessibility: Principles and guidelines - ARIA roles - Making sites accessible.
  • Project Work: Creating a complete website using HTML, CSS, Bootstrap and JavaScript/jQuery - Hosting the site online.
